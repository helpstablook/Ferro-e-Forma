# Ferro e Forma
Trabalho de coclusão de curso: O site do Ferro e Forma apresenta os especialistas em móveis industruais com peças exclusivas que unem ferro, madeira e design contemporâneo.

## Frontend:
- HTML (HyperText Markup Language): A estrutura básica de todas as suas páginas.

- CSS (Cascading Style Sheets): Usado para estilizar os elementos, principalmente através de classes.

- JavaScript (JS): Lógica do site (autenticação, carrinho, filtros, navegação entre checkout/login).

- Tailwind CSS: Framework utilitário de CSS carregado via CDN (Content Delivery Network) para design rápido (classes como bg-primary, flex, p-4).

- Google Fonts (Manrope): A fonte externa utilizada para o display do texto.

- Google Material Symbols: A biblioteca de ícones utilizada no projeto (ex: shopping_bag, dark_mode, chair).

## Backend/API:
- PHP: Responsável pela lógica de negócios, autenticação de usuários, processamento de formulários e interação com o banco de dados.

- MySQL (via PDO): Armazenamento de dados persistentes, como usuários (contas) e informações de produtos.

- Firebase Authentication: Serviço do Google para gerenciar o login e cadastro de usuários, autenticando via e-mail e senha.

- Firebase Realtime Database (ou Firestore): O banco de dados para armazenar os dados dos produtos e outras informações.

- Google Identity Toolkit (API): O endpoint para processar as requisições de login e emitir os tokens JWT.
